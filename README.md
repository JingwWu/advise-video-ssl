# INSTALL

```shell
# create conda env
conda create -n spd_py38 python=3.8
# install torch & torchvision

# install other pypi lib.
pip install -r requirements.txt
# download pytorchvideo source code from https://github.com/facebookresearch/pytorchvideo
# install pytorchvideo from source
cd /path/to/local/folder/pytorchvideo
pip install -e .
```
